PLANET RUNNERS

Mobile multiplayer racing game; which is heavily influenced by endless runner games, SpeedRunners and skill based MOBAs. I co-created Planet Runners with Burç Tuncer. He created all the visuals alongside with amazing soundtrack which you should definitely check out.

We aimed to create this game with "ULTRA MEGA SIMPLE CONTROLS" nevertheless with strong sense of control and fluidity. All heroes have different stats and various skills to overcome opponents.

Early in design we decided not to use any directional input. Players need to use basic controls (jump, dash) at the right time to indirectly control characters. This required a lot of tuning and balancing, meanwhile having a online multiplayer layer was a challenge.

Absence of directional input created interesting conditions. For example, creating a cyclic levels. We tried overcome this with arrow elements which changes the direction of the character and correctly placed springs.

VOODOO MAMBO

Being one of my first released games as a developer, this project holds relevancy. In this game, player uses its two fingers to merge balls on the touch screen with a pinch gesture.
It was essential to catch a feeling of ease at the pinch gesture. While gameplay required some motor skills, keeping the inference of basic physical failures to minimum was a clear aim we chased as a team. In retrospect, I believe we successfully realized the potential of the defined scope.
